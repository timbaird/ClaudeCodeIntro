# Claude Code Workshop

A hands-on, 4-hour workshop that teaches you how to use **Claude Code** — Anthropic's AI-powered coding assistant that runs directly inside VS Code.

## Important Note About AI Models

**Claude Code is designed to work with Anthropic's Claude models via a paid subscription, and will deliver significantly better results when used that way.** The paid Claude models (Sonnet, Opus) are purpose-built for the structured tool calling that Claude Code relies on — reading files, editing code, running commands, and managing projects.

In this workshop, we use a **free alternative via OpenRouter** so that students who do not wish to subscribe to Claude can still complete the exercises and learn the concepts. Be aware that:

- Free models will occasionally struggle with tool calls that paid Claude handles effortlessly
- You may need to rephrase requests or retry when the model gets confused
- This is a limitation of the free model, not of Claude Code itself
- If something isn't working, try simplifying your request or breaking it into smaller steps

**If you have access to a paid Claude subscription**, you can skip the OpenRouter setup and use Claude Code as intended — your experience will be smoother and more capable.

## Prerequisites

- A computer with internet access (Windows, macOS, or Linux)
- Basic familiarity with Python
- Basic familiarity with using a terminal/command line
- An OpenRouter account (free — set up in Exercise 0)

**No prior experience with Git, GitHub, or VS Code is required.** We'll walk you through everything.
