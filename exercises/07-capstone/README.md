# Exercise 7: Capstone Challenge

**Time: 15-30 minutes**

## The Challenge

Put everything you've learned together by building something of your own from scratch using Claude Code.

## Rules

1. **Start with a CLAUDE.md** — write it before any code
2. **Build iteratively** — one feature at a time
3. **Write tests** — ask Claude Code to test what it builds
4. **Review every change** — don't just auto-approve everything

## Choose Your Project

Pick **one** of the options below, or come up with your own idea of similar scope.

### Option A: Quiz App

Build a command-line quiz application:
- Stores questions and answers in a JSON file
- Presents questions one at a time
- Tracks the user's score
- Shows results at the end with percentage

### Option B: Contact Book

Build a command-line contact manager:
- Add contacts (name, phone, email)
- Search contacts by name
- Edit existing contacts
- Delete contacts
- Save to a JSON file

### Option C: Expense Tracker

Build a command-line expense tracker:
- Add expenses (amount, category, description, date)
- List all expenses
- Show total by category
- Show total for a given month
- Save to a JSON file

### Option D: Your Own Idea

Build anything you like! Keep it to a scope you can start in 15-30 minutes. Some ideas:
- A password generator
- A simple text-based game
- A file organiser
- A markdown note-taking tool

## Suggested Workflow

```
1. Choose your project
2. Write CLAUDE.md (project description, conventions, structure)
3. Ask Claude Code to create the data layer first
4. Ask Claude Code to write tests for the data layer
5. Ask Claude Code to create the CLI interface
6. Run and test manually
7. Ask Claude Code to add one extra feature
8. Run all tests
```

## Reflection Questions

When you're done (or when time is up), think about:

- How did writing CLAUDE.md first affect the code Claude generated?
- Did you review changes before approving them? Did you catch anything?
- How did you handle it when Claude Code made a mistake?
- What would you do differently next time?
- How could Claude Code help you in your other projects or coursework?
